Folder ini digunakan untuk menyimpan gambar-gambar portofolio Anda.
Pastikan nama file dan ekstensinya sesuai dengan yang ditulis di index.html dan projects.json:

1. profile.jpg -> untuk foto profil di header (ukuran persegi direkomendasikan).
2. project1.jpg -> untuk preview E-Commerce Website.
3. project2.jpg -> untuk preview Network Topology.
4. project3.jpg -> untuk preview Seminar Nasional Poster.
5. project4.jpg -> untuk preview Personal Blog.

Tips: Anda bebas mengubah nama file asalkan mengupdate baris yang sesuai di index.html (untuk profil) dan projects.json (untuk daftar project).
